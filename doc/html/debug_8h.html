task showstate ( void  )

Outputs the current state on the HT prototype board.

This tasks sends the current state to the digital outputs B of the HT prototype board. Depending on the definitions of the state the value can be differently interpreted. The current solution assumes a D6 circuitry implemented on the HT board with 7 LEDs that are linked as on a D6. The LEDs will be lit according to state. See the project NXT-D6 for building instructions. However, since only the state is sent as a byte to the HT board, any desired output circuitry can be used.
