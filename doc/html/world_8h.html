Detailed Description

Definitions and tasks about the world.

World definitions. World functions. World tasks. Everything related and dependant on the world the robot lives in should be defined here.

The mazes have the following properties:

  • rectangular grid of lines and junctions on 1m^2
  • colors of lines, background, junction, exit differ for different maze types

The Robot performs the following task in the world:

  • observe (observe the surface)

Note that depending on MAZE_TYPE definitions of constants and tasks differ.
