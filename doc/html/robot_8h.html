Detailed Description

Robot definitions and functions.

Robot definitions like input/output ports, speed, metrics and so on. Provides the function init which initializes all sensors.

The base model of the robot is the standard NXT Education Set model with slight modifications to the sensor array, which is built to house the HT color sensor parallel to surface. If the Lego color sensor should be used, this file needs to be adjusted.
